### ğŸ”¹ Pattern 2: Mathematical Calculations

```cpp

using namespace std;

    float radius = 5.0;
    float area = PI * radius * radius;
    cout << "Area: " << area << endl;
    return 0;
}
```

### ğŸ”¹ Pattern 3: Multiple Inputs

```cpp

    float length, breadth;
    cout << "Enter length: ";
    cin >> length;
    cout << "Enter breadth: ";
    cin >> breadth;
    cout << "Area: " << length * breadth << endl;
    return 0;
}
```
